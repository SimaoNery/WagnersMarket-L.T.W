### Wagner's Market
Welcome to the Wagner's Market.

#### Read Database
Go to the `private` folder and write:
```sh
cd private/
sqlite3 database.db
.read database.sql
```

#### Run
To run the project, go to the `public` directory:
```sh
cd public/
php -S localhost:9000
```
